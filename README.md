## Project Overview

Dubai Mall Digideck is a browser-based, interactive sales platform built to present a mega retail destination as a high-value commercial opportunity.

It exists to replace fragmented sales collateral—PDFs, decks, disconnected videos, and spreadsheets—with one immersive experience that communicates scale, momentum, and business potential in minutes.

The product is designed for prospective tenants, sponsors, and event partners evaluating where to invest, lease, or launch.

## Key Features

- Cinematic, video-first storytelling across the full journey
- Non-linear navigation that supports user-led exploration
- Immersive sections for Retail, Luxury, Attractions, and Events
- Dynamic engagement through hover states, click interactions, and transitions
- Conversion-focused CTA flow for leasing, partnerships, and event bookings

## Experience Philosophy

This is not a website, it is an experience designed to drive business decisions.

Every section is intentionally built to reduce friction between curiosity and action.

## Tech Stack

- Next.js
- Tailwind CSS
- Framer Motion
- AI-generated media (video + images)

## Architecture

The codebase follows a modular structure built for scale:

- `components/sections` for core narrative surfaces
- `components` for reusable UI building blocks
- `modules` for expandable domain flows and deeper product layers

This allows the experience to evolve without reworking the entire foundation.

## Performance Considerations

- Lazy loading for non-critical content
- Optimized media assets for faster delivery
- Smooth, controlled motion pipelines for consistent animation performance
